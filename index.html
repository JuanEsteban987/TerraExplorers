⛏️ Generador de Datapack / Mod

Crea un datapack para Minecraft Java Edition con recetas y funciones personalizadas.

Información del Datapack

Solo minúsculas, números y guión bajo. Sin espacios.

🍳 Recetas de Crafting

Cada receta es del tipo shapeless (sin forma).

⚙️ Funciones (comandos)

Cada función es un archivo .mcfunction con los comandos que escribas.

Los archivos generados son compatibles con Minecraft Java Edition. Coloca el .zip en la carpeta datapacks de tu mundo.
